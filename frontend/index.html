System Architecture
This object detection system uses AWS services with automated CI/CD deployment through GitHub Actions.
AWS Services
Amazon EC2
Flask API hosting
Amazon S3
Image storage
Amazon SageMaker
MobileNet v2 FPN-Lite inference
Amazon DynamoDB
Detection results storage
Amazon Bedrock
Nova Micro scene descriptions
IAM Roles
Secure service authentication
GitHub Actions
Automated CI/CD pipeline
Request Flow
1
User uploads image through web interface
2
Flask API on EC2 receives image, resizes it, and stores in S3
3
SageMaker endpoint processes image using MobileNet v2 FPN-Lite model
4
Bedrock Nova Micro generates natural language scene description
5
DynamoDB stores detection results for analytics
6
Results returned to frontend with detections and AI description
Technical Stack
Python 3.10 Flask + Gunicorn Boto3 SDK MobileNet v2 FPN-Lite PIL/Pillow JavaScript ES6 GitHub Actions CI/CD DynamoDB NoSQL Bedrock Nova Micro
