System Architecture
This object detection system uses a serverless AWS architecture with automated CI/CD deployment through GitHub Actions.
AWS Services
Amazon EC2
Flask API hosting
Amazon S3
Image storage & static hosting
Amazon SageMaker
MobileNet v2 FPN-Lite inference
CloudFront (Planned)
Global CDN distribution
IAM Roles
Secure service authentication
GitHub Actions
Automated CI/CD pipeline
Request Flow
1
User uploads image through web interface
2
CloudFront serves static frontend from S3
3
Flask API on EC2 receives image, resizes it, and stores in S3
4
SageMaker endpoint processes image using MobileNet v2 FPN-Lite model
5
Detection results returned to frontend with bounding boxes and confidence scores
Technical Stack
Python 3.10 Flask + Gunicorn Boto3 SDK MobileNet v2 FPN-Lite PIL/Pillow JavaScript ES6 GitHub Actions CI/CD SSH Deployment
