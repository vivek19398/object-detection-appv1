# Object Detection App — CS5024

An end-to-end object detection web application built on AWS, developed as part of **CS5024: Theory and Practice of Advanced AI Ecosystems** at the University of Limerick.

Users upload an image through the browser, the backend runs inference on a SageMaker-hosted model, Amazon Bedrock generates a natural-language description of the scene, and all results are stored in DynamoDB for later analytics.

---

## Features

- **Object detection** using MobileNet v2 FPN-Lite (COCO, 80 classes) deployed on Amazon SageMaker
- **Scene description** generated by Amazon Bedrock (Nova Micro) based on detected objects
- **Analytics dashboard** with detection history, stats, and per-object confidence, backed by DynamoDB
- **Image storage** in Amazon S3
- **CI/CD** with GitHub Actions — push to `main` automatically redeploys to EC2
- **Dark-themed single-page frontend** with drag-and-drop upload, live detection pills, and tabbed views
